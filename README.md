# PyLearn - Обучающий сайт по Python

Полнофункциональный интерактивный сайт для изучения основ Python на русском языке.

## 📁 Структура проекта

```
python-course/
├── index.html          # Основной HTML файл
├── style.css           # Стили сайта
├── script.js           # Основная логика и функциональность
├── lessons.js          # Контент уроков и вопросы теста
└── README.md           # Этот файл
```

## 🚀 Как запустить

### Способ 1: Локально (рекомендуется)

1. **Откройте файл `index.html` в браузере:**
   - Просто дважды щелкните на файл `index.html`
   - Или перетащите его в браузер
   - Или используйте Live Server в VS Code (установите расширение Live Server)

### Способ 2: С локальным сервером

Если у вас установлен Python:
```bash
# Python 3
python -m http.server 8000

# Затем откройте в браузере: http://localhost:8000
```

## 📝 Как редактировать контент

### Редактирование уроков

Откройте файл **`lessons.js`** и найдите массив `lessons`.

Каждый урок имеет структуру:
```javascript
{
    id: 1,
    title: 'Название урока',
    content: `Текст урока...`,
    examples: [
        'код примера 1',
        'код примера 2'
    ]
}
```

**Чтобы добавить новый урок:**
1. Скопируйте существующий урок
2. Измените `id`, `title`, `content` и `examples`
3. Добавьте в конец массива `lessons`

### Редактирование тестовых вопросов

Откройте файл **`lessons.js`** и найдите массив `questions`.

Каждый вопрос имеет структуру:
```javascript
{
    id: 1,
    question: 'Текст вопроса',
    options: ['Вариант A', 'Вариант B', 'Вариант C', 'Вариант D'],
    correct: 1,  // индекс правильного ответа (0-3)
    explanation: 'Объяснение правильного ответа'
}
```

**Чтобы добавить новый вопрос:**
1. Скопируйте существующий вопрос
2. Измените `id`, `question`, `options`, `correct` и `explanation`
3. Добавьте в конец массива `questions`

### Изменение стилей

Откройте файл **`style.css`** и отредактируйте CSS.

Основные CSS переменные (в начале файла):
```css
:root {
    --primary: #3b82f6;           /* Основной цвет (синий) */
    --secondary: #f97316;         /* Вторичный цвет (оранжевый) */
    --success: #22c55e;           /* Цвет успеха (зеленый) */
    --danger: #ef4444;            /* Цвет ошибки (красный) */
    --background: #f8fafc;        /* Цвет фона */
    --surface: #ffffff;           /* Цвет поверхности */
    --text: #1e293b;              /* Цвет текста */
    --text-light: #64748b;        /* Светлый текст */
}
```

Измените значения цветов, чтобы изменить внешний вид сайта.

### Изменение текста на главной странице

Откройте файл **`index.html`** и найдите нужный текст в разделе с id `home`.

## 🔧 Основные функции

### 📚 Уроки
- 5 полных уроков по основам Python
- Примеры кода для каждого урока
- Навигация между уроками

### 💻 Редактор кода
- Встроенный Python редактор
- Выполнение кода прямо в браузере (используется Pyodide)
- Вывод результатов в реальном времени
- Кнопки: Выполнить, Сброс, Копировать

### ✅ Тест
- 10 вопросов с множественным выбором
- Подробные объяснения ответов
- Результаты с процентом правильных ответов
- Возможность пересдачи теста

## 🐛 Решение проблем

### Редактор кода не работает
- Убедитесь, что у вас есть интернет (Pyodide загружается с CDN)
- Перезагрузите страницу
- Проверьте консоль браузера (F12) на ошибки

### Стили не применяются
- Очистите кэш браузера (Ctrl+Shift+Delete)
- Перезагрузите страницу (Ctrl+Shift+R)

### Синтаксические ошибки в коде
- Проверьте правильность JSON синтаксиса в `lessons.js`
- Убедитесь, что все кавычки и скобки закрыты
- Проверьте консоль браузера на ошибки

1. Откройте `lessons.js`

2. Добавьте новый объект в массив `questions`
3. Убедитесь, что индекс `correct` правильный (0-3)
